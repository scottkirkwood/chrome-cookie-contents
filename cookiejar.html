Cookies from:
  • LWP
  • Mozilla
  • Omniweb
    • The LWP format can easily be read by Perl and by Python programs. The format is slightly richer than the Mozilla format, dates that are easier to read.
    The Mozilla format used to be the way cookies were saved in Firefox, but is no longer the case (now it's stored in SqlLite database). You can read the Mozilla format in Python and in Perl. This format is tab delimited and doesn't support all of RFC 2965.
    The Omniweb format can be read by Perl.

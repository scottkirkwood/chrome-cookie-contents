    • The LWP format can easily be read by Perl and by Python programs. The format is slightly richer than the Mozilla format, and dates are easier to read.
    The Mozilla format used to be the way cookies were saved in Firefox, but is no longer the case (now it's stored in SQLite database). You can read the Mozilla format with Python and Perl. This format is not very extensible, is tab delimited and does not support all of RFC 2965.
    Omniweb is a web browser for the Macintosh. Their format is in XML and can be read by Perl.
